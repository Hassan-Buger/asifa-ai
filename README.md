## How to run (Windows)

### Step 1 — Install uv (one time only)
Open PowerShell and run:
```
powershell -c "irm https://astral.sh/uv/install.ps1 | iex"
```
Close and reopen PowerShell after this.

### Step 2 — Download the project
```
git clone https://github.com/Hassan-Buger/asifa-ai.git
cd asifa-ai
```

### Step 3 — Install everything
```
uv sync
```

### Step 4 — Run it
```
uv run rana_1
```
It will ask for your Gemini API key on first run.
Get a free key here → https://aistudio.google.com/app/apikey

## How to run (Mac/Linux)

### Step 1 — Install uv
```
curl -LsSf https://astral.sh/uv/install.sh | sh
```

### Step 2-4 — Same as Windows above
```
git clone https://github.com/YOURUSERNAME/asifa-ai.git
cd asifa-ai
uv sync
uv run rana_1
```
